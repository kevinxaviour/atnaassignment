# ATNA.AI Assignment

## Overview
Created a Python based data collection tool that:
- Scrapes quotes from a public website using HTML parsing with bs4
- Fetches Python repositories from the GitHub Search API
- Normalizes the collected data and exports it into a single JSON file
- Click here to view the [Final Output](https://github.com/kevinxaviour/atnaassignment/blob/c49be4838c01b762c328232d131b1d1338a950eb/output.json)
---

## How to Install and Run

### Install Dependencies
Make sure Python 3 is installed. Then install required libraries:

#### In bash
pip install -r requirements.txt

#### Run the main.py file
python main.py
